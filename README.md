# enigma-racing-my-stable

Enigma Racing MyStable shares management

## EN:

Development year: **April 2020**.

This is a project developed with the objective of keeping updated the horse shares of the subscribed members of [Enigma Racing](https://www.enigmaracing.net/), which came from a workbook generated in Excel that is used for it.

The main objective was to be able to update all this information by importing the workbook used without requiring any additional action by the administrator, thus keeping the database of the shares of all members updated every time it was required in a short time.



## Languages/Technologies used:

- HTML.

- CSS.

- JS.

- PHP.

- JQuery.

- Bootstrap.



## Project functionalities:

- Login/registration system.
- Dashboard of the members to visualize their shares invested in each horse.

- Administrative Dashboard to visualize all the members and stables that are registered, as well as the payments made by each member.
  - Function to update the stock database through the workbook every time it is required.
    - Each time the workbook is uploaded, each page of the workbook will be scanned to extract all the data that is required to update the stock database.

- User management for the administrator.



## ES:

Año de desarrollo: **Abril 2020**.

Este es un proyecto desarrollado con el objetivo de mantener actualizadas de las acciones en caballos de los miembros suscritos de [Enigma Racing](https://www.enigmaracing.net/), las cuales provenían de un workbook generado en Excel que se utliza para ello.

El objetivo principal era poder actualizar toda esa información mediante la importación del workbook utilizado sin requerir ninguna acción adicional por parte del administrador, haciendo de esta manera, mantener actualizada la base de datos de las acciones de todos los miembros cada vez que se requiriera en poco tiempo.



## Lenguajes/Tecnologías usadas:

- HTML.

- CSS.

- JS.

- PHP.

- JQuery.

- Bootstrap.



## Funcionalidades del proyecto:

- Sistema de login/registro.

- Dashboard de los miembros para visualizar sus acciones invertidas en cada caballo.

- Dashboard administrativo para visualizar todos los miembros y establos que se encuentren registrados, así como los pagos realizados de cada miembro.
  - Función para actualizar la base de datos de las acciones mediante el workbook cada vez que se requiera.
    - Cada vez que el workbook es subido, cada página de él será escaneada para extraer todo los datos que son requeridos para la actualización de la base de datos de las acciones.

- Gestión de usuarios para el administrador.
